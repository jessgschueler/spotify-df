# Spotify Data Exploration

### by Jess Schueler

#### *Utilizing Pandas to intake, clean and explore Spotify data.*

## Technologies Used
* Python
* Pandas
* Jupyter Notebook

## Description 
A Jupyter Notebook that intakes multiple csv files of Spotify artist, album, and track data. Utilizes Pandas to read, explore, clean, merge, and interpret the data in multiple data frames.

## Setup/Installation Requirements
* In the terminal, clone github repository using the following command;
```
$ git clone https://github.com/jessgschueler/spotify-df
```
* In a venv, Pip install requirements.txt file
* Extract the data using the get_data.sh file
* Open the main.ipynb file and run the code cell
